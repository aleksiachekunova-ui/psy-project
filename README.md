# psy-project  
Fill Your Cup — an AI-driven mental health companion.

## Overview  
Fill Your Cup is an emotionally intelligent digital wellbeing tool designed to support daily reflection, emotional regulation, and long-term mental resilience. The project integrates evidence-based psychological practices with an advanced empathetic AI model capable of adapting to individual users.

## Vision  
The goal of this project is to create a calm, supportive, and scientifically grounded environment where users can:  
- engage in structured, meaningful self-reflection  
- build emotional awareness and resilience  
- track behavioral and emotional dynamics  
- receive adaptive insights generated by an empathy-oriented AI system  

## Key Features  
- **AI-Guided Emotional Journal** — daily reflection supported by contextual prompts  
- **Empathy-Driven AI Feedback** — personalized, context-aware responses  
- **Pattern Recognition and Trend Analysis** — detection of emotional and behavioral shifts  
- **Micro-Practices for Stress Regulation** — concise evidence-based wellbeing exercises  

## Technology Stack  
- Python  
- FastAPI  
- SwiftUI (iOS)  
- Custom empathetic AI architecture  
- Lightweight personalization and behavioral modeling  

## Research Focus  
The project explores:  
- computational modeling of empathic responses  
- insight-based (non-linear) learning in emotional AI  
- cognitive-emotional pattern extraction from short-form journaling  
- applications of affective computing in daily wellbeing  

## Current Status  
The system is in active development. AI components are being refined to improve emotional attunement, contextual reasoning, and personalized guidance. User experience is evolving through iterative research and prototyping.
